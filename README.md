SoundSnap is a mobile game for children. During the game, sounds are played. The user earns points for matching the image to the sound heard. Each game is time-limited.
<br>
<br>The application was created with:
<br>Kotlin
<br>Jetpack Compose
<br>Hilt
<br>MVI architecture
